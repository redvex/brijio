User-controlled browser bridge for AI agents

Brijio

Brijio gives AI agents precise, request-based access to the browser the user has explicitly connected. It is built for demos that need real page context, real DOM actions, and clear user control.

Built for controlled access

Brijio is reactive. Browser state is requested through explicit MCP calls, routed over a WebSocket channel, and returned as structured results.

User starts the bridge

The extension connects only after a user action and can be disconnected any time.

Agents make explicit calls

Every read, click, fill, navigation, fetch, or download action starts as a tool request.

Sessions stay private

Routing is designed around user, session, and browser instance boundaries.
