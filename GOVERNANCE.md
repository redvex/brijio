# Governance

## Project Status

Brijio is currently a founder-led open source project.

The project is maintained by the Brijio maintainers and contributors.

## Principles

- User ownership over infrastructure
- Transparency by default
- Security-first engineering
- Open protocols
- Sustainable open source development

## Decision Making

Significant architectural decisions should be documented through ADRs.

Maintainers are responsible for final technical decisions.

Community feedback is encouraged through issues, discussions, and pull requests.

## Maintainers

Initially, the project maintainer is the Brijio founder.

Additional maintainers may be appointed based on sustained contribution and community trust.

## Roadmap

The roadmap is maintained publicly.

The maintainers reserve the right to prioritize work based on project sustainability, security requirements, and le resources.

## Governance Evolution

This document may evolve as the project grows.
