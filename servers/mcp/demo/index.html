User-controlled browser bridge for AI agents

Brijio

Brijio gives AI agents precise, request-based access to the browser the user has explicitly connected. It is built for demos that need real page context, real DOM actions, and clear user control.

Built for controlled browser access

The extension is reactive: browser state is requested through MCP tools, routed over WebSocket, and returned as structured results. No continuous page streaming is required for this demo.

User starts the bridge

The browser connects only after the user chooses to connect from the extension UI.

Agents make explicit calls

Every read, click, fill, navigation, fetch, or download action starts as a clear MCP tool request.

Sessions stay private

Routing is designed around user, session, and channel identifiers for local and future cloud deployments.
