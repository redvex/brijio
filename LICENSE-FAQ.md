# Brijio License FAQ

## What license is Brijio released under?

Brijio is licensed under AGPLv3 (GNU Affero General Public License v3.0). See the [LICENSE](LICENSE) file for the full text.

## What does AGPLv3 mean for me?

- **Use**: You can run Brijio for any purpose, including commercially.
- **Study**: You can read and modify the source code.
- **Distribute**: You can redistribute the original or modified versions.
- **Network use**: If you offer Brijio as a network service, you must make the source code available to users of that service.

## Do I need to open-source my modifications?

If you distribute Brijio (including over a network), you must make the modified source code available under AGPLv3. This applies to the Brijio code itself, not your separate applications that merely connect to it.

## What if I want to use Brijio in a proprietary product?

Organizations that wish to embed Brijio into proprietary products or offer Brijio-based services without AGPLv3 obligations may obtain a separate commercial license. See [COMMERCIAL-LICENSING.md](COMMERCIAL-LICENSING.md) for details.
